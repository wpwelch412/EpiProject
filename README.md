# EpiProject - Pediatric Epilepsy Surgery Assessment

This repository contains a comprehensive web application for evaluating pediatric epilepsy surgery candidacy.

## Overview

The Pediatric Epilepsy Surgery Assessment Tool is designed to help patients, families, and healthcare providers evaluate potential candidacy for epilepsy surgery. The tool provides:

- Structured questionnaire covering key clinical domains
- Evidence-based assessment of surgical candidacy
- Detailed information about appropriate surgical options
- Recommendations for next steps in evaluation

## Application

The main application is in the `epilepsy-assessment` directory. See the [full documentation](./epilepsy-assessment/README.md) for details.

### Quick Start

```bash
cd epilepsy-assessment
npm install
npm run dev
```

## Features

- **Client-side only** - No data storage, completely private
- **Comprehensive assessment** - Evaluates seizure characteristics, medication history, diagnostic workup, and demographics
- **Evidence-based** - Built on established epilepsy surgery candidacy criteria
- **Professional output** - Print-friendly results for sharing with medical teams
- **Mobile responsive** - Works on all devices

## Medical Disclaimer

This tool is for educational and screening purposes only. It does not replace comprehensive medical evaluation by qualified epilepsy specialists. All treatment decisions should be made in consultation with your healthcare team.

## Technology

Built with React and Vite for a modern, fast, and accessible user experience.
